This is a calculator for the Grothendieck group of a repetitive cluster category of Dynkin type A or D, based on a paper by Chang--Murphy--Zhou.

The variable n refers to the number of vertices in the respective quiver (alternatively, number of simple modules over the path algebra).

The variable p refers to the power of autofunctor \tau \Sigma^{-1} in the definition of a repetitive cluster category.

All efforts have been made to resolve any edge cases, however please contact Dave Murphy if any discrepencies are noticed.
